🎬 MoodMatch
MoodMatch is a mood-based movie recommender web app built with Flask and vanilla JavaScript.
Pick how you're feeling — happy, sad, excited, scared, bored, or romantic — and get an instant handpicked film recommendation. Reshuffle for a different pick, browse the full curated library, or check your session history.
Features

6 moods × 4 handpicked films each
Random recommendation engine with session history
Full film library browser
JSON API at /api/movies/<mood>
Single-page frontend — no page reloads

Stack

Backend — Python / Flask
Frontend — HTML, CSS, JavaScript (single file)
